About Savr

Built because grocery apps
shouldn't make you do maths.

The problem

Blinkit, Zepto, Instamart, BigBasket and JioMart all sell the same packet of milk — but at five different prices, with five different delivery fees, and sometimes five different promotions running at the same time. To get the cheapest option, you'd have to open every app, search for the same product, compare numbers in your head, then close four of them. Every. Single. Purchase.

Nobody has time for that.

What Savr does

Savr is a single search box that simultaneously queries all five major quick-commerce platforms in India. You type the product name once. We scrape live prices from every platform in parallel using headless browser automation, then return a flat sorted list showing you which platform has it cheapest, right now.

No account. No login. No dark patterns. Just prices.

How the tech works

The backend is a Node.js server running five Puppeteer scrapers in parallel — one for each platform. Each scraper handles location gating, lazy loading, and the unique DOM structure of its target site. Results are returned as a flat JSON array and rendered live in the browser.

Prices are fetched fresh on every search — not from a cache, not from a database. What you see is what's live on the platform right now.
